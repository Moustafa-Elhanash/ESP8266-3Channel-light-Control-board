3-Channel Light Control PCB
Overview

This project showcases a 3-channel light control PCB designed for home automation applications. The board is built around the ESP8266 microcontroller and allows users to control three separate lights through any compatible mobile application. This repository focuses on the hardware design, including the schematic and PCB layout, highlighting the designer’s work in creating a functional and flexible PCB solution.

Features

3 independent relay channels for light or appliance control

ESP8266-based wireless control

Designed to be compatible with any mobile application supporting relay switching

Compact PCB layout optimized for home installation

Complete schematic and layout files included

Files Included

schematic/ – Folder containing the complete schematic diagrams

PCB/ – Folder with the PCB layout files

README.md – This description and project overview

Purpose

This repository demonstrates PCB design skills, showcasing the ability to:

Plan and organize multi-channel control circuits

Design a compact and functional PCB

Create hardware that is flexible and ready to integrate with mobile control applications
